# Surya LinkedIn AI Agent

LinkedIn‑AI‑Agent is a lightweight automation toolkit for publishing thoughtful LinkedIn posts using AI. It orchestrates the entire content cycle—from topic selection through generation, storage, and scheduling—to keep your feed active and optimized.

This repository hosts the `Libkedin-AI-Agent` package containing modular components such as topic management, content generation, scheduling, and publishing. Use this toolkit to prototype or run an automated LinkedIn content pipeline.

## Getting Started

1. Create a virtual environment: `python -m venv venv`
2. Activate it: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Unix)
3. Install dependencies: `pip install -r Libkedin-AI-Agent/requirements.txt`
4. Run the agent: `python Libkedin-AI-Agent\main.py`

Refer to `Libkedin-AI-Agent/README.md` for more details on individual components.
